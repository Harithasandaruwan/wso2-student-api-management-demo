# WSO2 Student API Management Demo

This project demonstrates how a simple Node.js Student API can be published, secured, subscribed to, and invoked through WSO2 API Manager.

## Project Flow

Frontend App → WSO2 API Gateway → Node.js Backend API

## Technologies Used

- Node.js
- Express.js
- WSO2 API Manager 4.7.0
- WSO2 Developer Portal
- WSO2 Publisher Portal
- Postman
- HTML, CSS, JavaScript
